# zh-site-lab

一个用于归档和发布多个独立 HTML 页面的仓库。所有页面均为静态内容，不依赖后端服务，便于直接浏览或部署。

## 项目简介

本仓库主要存放各类独立的 HTML 页面，这些页面可能用于展示、演示、测试或其他非商业用途。每个页面独立存在，互不依赖，适合作为前端素材的存档或轻量级展示页面的集合。

## 目录说明

```
/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 归档的独立 HTML 页面
│   ├── example1.html
│   └── example2.html
└── README.md           # 本文件
```

- `pages/` 目录用于存放各个独立的 HTML 页面文件。
- 每个页面应保持自包含，避免引用外部资源（除非资源稳定可访问）。
- 可根据需要创建子目录进行分类归档。

## 页面归档说明

- 所有页面均为静态 HTML，不包含动态内容或后端逻辑。
- 页面命名建议使用英文小写字母、数字及连字符，避免特殊字符。
- 如需引用图片、样式或脚本，请尽量内联或使用相对路径。
- 归档内容不针对任何特定网站或域名，仅为技术存档用途。

## 维护说明

- 欢迎提交新的独立页面，但请确保内容不违反相关法律法规及平台政策。
- 提交前请检查页面在不同浏览器下的基本兼容性。
- 不建议在页面中添加跟踪、广告或恶意代码。
- 本仓库仅作为归档和发布平台，不对页面内容的准确性或时效性做保证。

## 使用方式

可通过以下方式浏览本仓库中的页面：

1. 直接克隆仓库到本地，用浏览器打开对应 HTML 文件。
2. 部署到任何静态托管服务（如 GitHub Pages、Netlify 等）后在线访问。
